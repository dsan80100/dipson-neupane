# Dipson Neupane - Portfolio Website

## Overview
A modern, responsive portfolio website built with React and Vite for Dipson Neupane. The website showcases professional work, services, and provides contact information.

## Project Structure
```
├── public/
│   └── assets/          # Images (profile, about, projects)
├── src/
│   ├── components/      # React components
│   │   ├── Navbar.jsx/css
│   │   ├── Home.jsx/css
│   │   ├── About.jsx/css
│   │   ├── Services.jsx/css
│   │   ├── Projects.jsx/css
│   │   ├── Contact.jsx/css
│   │   └── Footer.jsx/css
│   ├── App.jsx
│   ├── App.css
│   ├── main.jsx
│   └── index.css
├── index.html
├── vite.config.js
└── package.json
```

## Features
- **Home Section**: Hero section with profile image, introduction, and call-to-action buttons
- **About Section**: Personal information, skills with progress bars, and statistics
- **Services Section**: 6 service cards showcasing offered services
- **Projects Section**: Recent work gallery with 6 project showcases
- **Contact Section**: Contact form and social media links
- **Footer**: Site navigation, social links, and contact information

## Technologies
- React 18.3.1
- Vite 5.4.2
- CSS3 with custom animations
- Fully responsive design
